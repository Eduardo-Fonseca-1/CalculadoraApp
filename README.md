O aplicativo Calculadora Simples permite ao usuário realizar operações matemáticas básicas de forma rápida e intuitiva.

Funcionalidades principais:

Adição
Subtração
Multiplicação
Divisão

O app possui uma interface simples, com botões numéricos e operadores, facilitando o uso por qualquer usuário.
